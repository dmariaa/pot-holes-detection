# Pothole detection

Pipeline and tools for loading mobile sensor sessions, merging labels, and
generating spectrogram samples for pothole detection research.

## Quickstart (uv)
```powershell
uv sync
uv run python -m potholes.loader --help
```

## Documentation
Project docs live in `docs/` and are served with MkDocs:

```powershell
uv run mkdocs serve --livereload
```
