# CLI Usage

Use the CLI tools from the repository root. Each tool has its own help page:

- Loader: `cli/loader.md`
```powershell
python -m potholes.loader --help
```
- Plotter: `cli/plotter.md`
```powershell
python -m potholes.tootls.plot --help
```
- Video: `cli/video.md`
```powershell
python -m potholes.tools.video --help
```
